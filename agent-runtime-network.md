# Agent Runtime Network

运行层：治理智能体运行网络

目标：展示多个治理 Agent 正在协同工作。

## 1. 核心结构

Agent Runtime Network 由六个智能体节点组成：

- 风险感知 Agent
- AI 治理研判 Agent
- 政策解析 Agent
- 社区协同 Agent
- 案例学习 Agent
- 未来推演 Agent

## 2. 运行态

必须展示：

- Agent 在线。
- Agent 正在处理任务。
- Agent 之间存在信号流。
- 治理事件正在跨 Agent 传递。
- 结果进入城市记忆。

## 3. 可视化规则

每个 Agent 拥有：

- 独立光场。
- 独立脉冲。
- 独立扫描方式。
- 独立信号颜色。
- 独立协同轨迹。

但必须共享同一视觉文明：

- 深空背景。
- 灰调青绿色主色。
- 低饱和 Amber 风险色。
- 慢速呼吸动效。

## 4. 禁止

- 机器人头像。
- AI 助手拟人化。
- 聊天窗口主导。
- NPC 卡片。
- 科幻角色设定。

## 5. 第一版实现

第一版使用 mock runtime data：

- 风险感知 Agent 识别公共收益争议。
- AI 治理研判 Agent 生成摘要。
- 政策解析 Agent 匹配治理规则。
- 社区协同 Agent 建议多主体协作。
- 案例学习 Agent 匹配历史案例。
- 未来推演 Agent 预测风险扩散路径。
