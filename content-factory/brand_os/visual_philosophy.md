# Visual Philosophy｜视觉哲学

## 视觉总原则

不要设计高级感。

要让它像一种天然存在的公共秩序。

网站、视频、封面、海报、PPT 都应该像：

- 档案馆
- 市政大厅
- 公共图书馆
- 治理研究机构
- 社区公共空间
- 一部关于信任的行业纪录片

## 摄影语言

焦段：

- 35mm 为主，用于真实治理现场。
- 50mm 用于账本、纸张、手部动作、印章、文件特写。
- 少量 24mm 用于空间远景。
- 禁止超广角变形。

景深：

- 不做广告级浅景深。
- 保留环境层次。
- 人、空间、物件共同存在。

构图：

- 大量留白。
- 中心偏移。
- 人物不占据画面中心。
- 允许空场。
- 空椅子、桌面、走廊、公告栏都可以成为主角。

摄影机运动：

- 固定镜头为主。
- 极慢推镜。
- 少量滑轨。
- 禁止快节奏运动。

曝光：

- 略压暗 0.3EV。
- 保留阴影细节。
- 不做高亮商业曝光。

颗粒：

- 允许轻微真实颗粒。
- 不要干净到像 CG。

人物规则：

- 不直视镜头。
- 不职业笑容。
- 不摆拍会议感。
- 更像真实治理现场。

## 光影

使用：

- 清晨灰度。
- 阴天漫射光。
- 室内侧光。
- 纸张边缘柔光。
- 走廊尽头低亮度光。

禁止：

- 蓝紫科技光。
- 大面积发光渐变。
- 粒子光效。
- 商业宣传片高光。

## 材质

核心材质：

- 纸
- 亚麻
- 木
- 石材
- 磨砂玻璃
- 亚光金属
- 文件柜
- 公告栏亚克力

材质必须有寿命。

纸会变旧。

铜会氧化。

木会留下使用痕迹。

时间感才会产生信任感。

## 品牌圣物

共同资产治理账本。

它不是道具，不是 icon，不是装饰。

它是信任被记录下来的证据。

出现路径：

1. 半遮挡。
2. 被打开。
3. 翻到预算页。
4. 与账户结构并置。
5. 合上并归档。

## 去互联网化

减少：

- 按钮
- 卡片
- 图标
- Dashboard
- 标签
- 产品卖点
- UI 炫技

增加：

- 空间
- 档案
- 记录
- 时间
- 材质
- 真实物件
- 公共场景
- 克制判断
