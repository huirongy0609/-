# Conversion Funnels｜转化漏斗系统

目标：把内容流量变成可跟进线索，再把线索变成成交。

## 标准漏斗

```text
内容
↓
关注
↓
评论 / 收藏
↓
私信
↓
资料 / 社群
↓
咨询
↓
成交
↓
复购 / 转介绍
```

## 公众号漏斗

1. 长文建立信任。
2. 文末引导领取资料或预约诊断。
3. 私信自动回复资料。
4. 邀请进入社群。
5. 社群持续交付内容拆解。
6. 直播或一对一咨询转化。

## 小红书漏斗

1. 标题承接搜索。
2. 正文给清单和场景。
3. 评论区引导“想要模板可以留言”。
4. 私信发送资料。
5. 询问用户账号阶段和问题。
6. 推荐小课、训练营或咨询。

## 视频号漏斗

1. 口播建立真人信任。
2. 评论区放关键词。
3. 引导看公众号长文。
4. 引导进直播。
5. 直播间转化课程或服务。

## 抖音漏斗

1. 强钩子拉新。
2. 评论区筛需求。
3. 私信承接资料。
4. 引导到微信或社群。
5. 用案例和直播做转化。

## 私域漏斗

1. 用户加微信。
2. 发送欢迎语和资料。
3. 询问当前阶段。
4. 打标签：新手 / 有账号 / 有产品 / 团队 / 高客单。
5. 推送对应内容。
6. 邀请咨询或直播。
7. 成交后进入交付群。

## 漏斗关键指标

| 阶段 | 指标 | 目标 |
|---|---|---|
| 内容曝光 | 阅读 / 播放 | 判断标题和选题 |
| 内容互动 | 点赞 / 收藏 / 评论 | 判断共鸣和价值 |
| 私信 | 私信数 | 判断需求强度 |
| 社群 | 入群数 | 判断承接效率 |
| 咨询 | 预约数 | 判断信任强度 |
| 成交 | 订单数 / 金额 | 判断商业化效率 |

## 漏斗优化原则

- 阅读高但私信低：内容有传播，转化入口弱。
- 收藏高但成交低：方法有价值，但信任不足。
- 评论高但私信低：评论区引导不够明确。
- 私信高但成交低：咨询脚本或产品阶梯有问题。
- 成交高但曝光低：应该矩阵改写放大。
