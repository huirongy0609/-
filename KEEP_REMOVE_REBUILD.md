## 3. 重构项

### 首页

当前：全国信托制社区治理协同平台展示页。

目标：信托制社区治理平台统一入口。

重构原因：

- 当前仍偏“平台呈现”，但缺少聚道研究院、金牌管家成长世界的品牌架构表达。
- 首页应从数字态势转为内容、制度、成长、资源连接的入口。

### AI情报页

当前：AI 情报 + Agent 运行空间。

目标：AI信托助手。

重构原因：

- “AI中心 / AI情报中心 / Agent 运行空间”不符合新定位。
- 应改为“找制度、找内容、找案例、找模板、找课程、生成材料”的助手型入口。

### 案例库

当前：独立案例库。

目标：内容中心 > 实践案例。

重构原因：

- 新规划明确案例不再独立建设案例中心。
- 案例应成为内容中心的一类内容。

### 地图页

当前：城市运行 / 协同网络 / 风险视觉。

目标：资源连接或治理实验室辅助入口。

重构原因：

- 新平台不是治理驾驶舱或城市运营平台。
- 地图/网络不能成为核心产品心智。

### 共建提交页

当前：提交案例、政策线索、城市动态等。

目标：用户识别与资源连接系统。

重构原因：

- 用户中心不应是普通栏目，而应先完成用户画像识别。
- 用户画像包括身份、城市、单位、当前状态、关注方向。
- 内容推荐、成长任务推荐、资源连接推荐应基于画像展示。
- 当前表单只是一个粗入口，缺少用户识别前置流程。

### README 与文档入口

当前：旧 MVP 说明。

目标：信托制社区治理平台说明。

重构原因：

- 当前 README 与冻结品牌定位不一致。

### CSS

当前：`app/globals.css` 超过 3700 行，混合多阶段样式。

目标：保留基础视觉，分层清理。

重构原因：

- 当前 CSS 是最大熵源。
- 多阶段样式共存，容易导致视觉漂移。

## 5. 判断原则

保留：能服务内容、制度、成长、用户、人才资产的内容。

删除：明显服务旧“驾驶舱/宇宙/城市运行”方向且未被引用的内容。

重构：有价值但定位不准确的页面、组件和数据。
