# Approval Rules

Status: Draft
Version: v0.1.0
Date: 2026-07-12

## 文件状态

所有正式文件至少区分：

- Draft：草稿。
- Review：审核中。
- Approved：已批准。
- Superseded：已被新版本替代。
- Archived：已归档。

## 已批准文件保护

已经批准的文件，Codex 不得直接修改正文。

如需调整，必须：

1. 新建分支。
2. 提交变更说明。
3. 形成 Pull Request。
4. 标明影响范围。
5. 等待审核批准。

## 审批责任

- 内容口径、标准、制度、专业定义：Work 编制或修改，ChatGPT 审核，杨慧荣批准重大事项。
- 网站工程、脚本、数据结构、部署：Codex 执行，ChatGPT 审核影响范围。
- 重大方向、阶段进入、争议裁决：杨慧荣批准。

## 禁止直接修改

除非任务明确批准，不得直接修改：

- 书稿或引用源文档。
- 已批准标准。
- 信托制物业核心定义。
- 已发布专业知识正文。
- 影响全站结构的底层目录。
