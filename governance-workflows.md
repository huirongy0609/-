# Governance Workflows

治理工作流描述系统如何处理真实社区事件。

## 1. 停车纠纷工作流

事件出现 → 风险感知 → 舆情识别 → AI 摘要 → 案例匹配 → 街道协同 → 物业响应 → 业委会参与 → 居民沟通 → 风险缓解 → 案例沉淀。

## 2. 公共收益争议工作流

线索提交 → 收益账户核验 → 财务公开风险识别 → 政策规则匹配 → 社区议事 → 业主代表确认 → 公开说明 → 信任修复 → 透明治理案例沉淀。

## 3. 物业服务冲突工作流

投诉聚合 → 高频问题归类 → 服务合同核对 → 物业响应要求 → 社区协调 → 专家建议 → 整改跟踪 → 满意度回访。

## 4. 舆情风险工作流

舆情升温 → 关键词聚类 → 情绪判断 → 扩散路径识别 → AI 摘要 → 街道关注 → 统一回应 → 线下沟通 → 舆情降温。

## 5. 老旧小区改造协同工作流

问题征集 → 居民意愿汇总 → 政策资金匹配 → 改造方案共议 → 物业/施工/社区协同 → 风险提示 → 阶段反馈 → 经验沉淀。

## 输出要求

每条工作流都需要呈现：

- 当前事件
- 风险等级
- 已参与角色
- AI 辅助结果
- 下一步治理动作
- 结果状态
