# City Governance Dashboard

定位：城市级社区治理操作台。

目标：展示城市治理态势、风险、AI研判、协同效率和案例网络。

## 1. 页面气质

像城市运行控制台，而不是后台管理系统。

关键词：

- 全局态势
- 风险研判
- 协同状态
- AI 分析
- 案例沉淀
- 城市覆盖

## 2. 主界面结构

```text
┌──────────────────────────────┐
│ Command Header               │
├───────────────┬──────────────┤
│ Map System    │ AI Panel      │
│               │ Risk Queue    │
├───────────────┴──────────────┤
│ Key Metrics / Trend Strip     │
└──────────────────────────────┘
```

## 3. 核心模块

### 全国 / 城市地图

展示：

- 城市节点
- 社区覆盖
- 风险热区
- 协同连接线
- 共建状态

### AI治理助手

展示：

- 政策摘要
- 风险分类
- 舆情总结
- 纠纷研判
- 相似案例
- 建议动作

### 风险队列

展示：

- 事件类型
- 区域
- 风险等级
- 触发信号
- 处理建议

### 指标栏

展示：

- 社区数量
- 风险事件
- 物业评分
- 协同效率
- 舆情趋势
- AI分析结果

## 4. 信息优先级

1. 地图态势
2. 风险状态
3. AI建议
4. 关键指标
5. 案例与政策

## 5. HUD规则

- HUD 只显示必要信息。
- 线条细，透明度低。
- 面板边框克制。
- 不使用复杂仪表盘装饰。
- 风险提示不做红色警报灯。

## 6. 典型镜头

### 全景

大屏中显示全国地图，右侧 AI 面板，底部指标条。

### 下钻

从全国地图进入一个城市，再进入社区网格。

### 风险识别

单个小区出现 Amber 风险点，AI 扫描后弹出分类。

### 协同处置

风险事件连接到案例库、政策摘要、共建机构。

## 7. 文案风格

使用：

- 风险识别
- 协同状态
- 研判完成
- 案例匹配
- 建议进入协同处置

避免：

- 震撼
- 颠覆
- 一键解决
- 黑科技
- 爆款
