# Map System Guide

地图系统是社区治理协同平台的核心入口。

目标：用地图呈现城市、社区、小区、事件、风险和协同状态。

## 1. 地图定位

地图不是装饰。

地图用于回答：

- 哪些城市已接入？
- 哪些社区有治理案例？
- 哪些区域出现风险？
- 哪些城市存在相似问题？
- 哪些经验可以跨城市复用？

## 2. 地图层级

```text
Level 1 全国轮廓
Level 2 省市暗纹
Level 3 城市节点
Level 4 社区 / 小区节点
Level 5 风险热区
Level 6 数据流与协同线
Level 7 城市 / 事件详情卡
```

## 3. 城市节点

### 已点亮城市

- Civic Teal
- 小光晕
- 可点击
- 显示最近动态

### 待加入城市

- Data Gray
- 低透明度
- 不高亮

### 风险关注城市

- 主节点仍保持青绿色
- 旁边增加 Amber 小标识
- 不覆盖城市状态

## 4. 风险热区

风险热区用于提示治理关注，不制造灾难感。

规范：

- 低饱和 Amber
- 弱扩散圈
- 不大面积染红
- 不闪烁
- 不震动

## 5. 数据流

数据流表示：

- 案例复用
- 政策同步
- 城市共建
- AI情报流转

规范：

- 1px 细线
- 低透明度
- 少量粒子
- 速度慢
- 不做流星雨

## 6. 地图标签

标签示例：

- 信托制物业试点
- 老旧小区治理热区
- 公共收益透明治理
- 舆情风险扫描
- 城市共建节点

标签规范：

- 小字号
- 深色玻璃底
- 青绿色边框
- 不遮挡主地图

## 7. 地图动效

允许：

- 节点点亮
- 弱扫描环
- 细线连接
- 风险弱扩散
- 城市卡片展开

禁止：

- 3D 地球
- 快速旋转
- 大面积粒子
- 红色警报
- 炫光扫屏

## 8. 地图镜头语言

### Pull Back

小区 -> 社区 -> 城市 -> 全国。

### Drill Down

全国 -> 城市 -> 社区 -> 风险点。

### Lateral Pan

横向浏览城市卡片和协同网络。

### Static View

用于结尾和关键判断，建立可信度。
