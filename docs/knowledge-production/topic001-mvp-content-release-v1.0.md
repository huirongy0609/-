# 一、Topic首页展示内容

## 1.1 Topic标题

**共同委托**

## 1.2 一句话简介

看懂业主如何把共同决定转化为有边界、受监督的专业执行。

## 1.3 导读

住宅小区里的保洁、维修、秩序维护、资金使用和合同执行，需要持续的专业服务；但涉及共有部分和共同利益的重大事项，不能由任何个人单独决定。“共同委托”要回答的，就是业主如何通过共同决定和有效授权，把特定事务交给专业主体执行，同时保留决定权与监督权。

理解共同委托，需要同时看清四件事：管理权限从哪里来，专业主体可以做什么，应当承担什么责任，出现问题或更换服务人时如何退出和交接。它不是每位业主个人委托的简单相加，也不是把共同管理权全部交出去，更不当然等同于《信托法》意义上的信托。阅读本专题，可以从共同决定、受托关系、信义义务和开放式预算等基础概念出发，建立一条从权利、授权到履职监督的清晰路径。

# 五、Topic页面第一屏介绍

住宅小区的共同事务，需要业主依法形成共同决定，也需要专业主体持续执行。共同委托连接的正是这两个环节：它让管理权限有清楚来源，让服务人知道可以做什么、应当承担什么责任，也让业主保留知情、监督、纠偏和更换的权利。

阅读《共同委托》，不是为了记住一个新名词，而是为了回答社区治理中最关键的几个问题：谁有权决定，谁可以执行，权限边界在哪里，资金和服务如何公开，发生分歧时怎样处理，关系终止后如何交接。理解这条从共同决定到受托履职的责任链，是继续认识信托制物业、信义义务和开放式预算的基础。
