# CURRENT_TO_TARGET_MAP

本文件用于把当前项目实际结构映射到新的冻结定位：`信托制社区治理平台`。

## 1. 品牌定位映射

| 当前状态 | 目标状态 | 处理方式 |
|---|---|---|
| 页面 metadata：全国信托制社区治理协同平台 | 平台名称：信托制社区治理平台 | 重命名，不另起项目 |
| README：全国社区治理协同地图 MVP | 信托制社区治理平台 | 后续重写 README |
| 首页：全国信托制社区治理协同平台 | 平台统一入口 | 保留首页，但重写信息结构 |
| 停车纠纷 Demo | 治理实验室 / 治理案例样板间 | 保留，降级为实验室样板，不作为平台主线 |
| AI情报 / Agent 运行空间 | AI信托助手 | 重构命名与职责，降级为全站能力 |
| 案例库 | 内容中心下的“实践案例”栏目 | 合并进内容中心，不再独立为一级案例中心 |
| 首页“成长认证中心”与产品化内容混杂 | 成长世界 + 产品中心 | 成长训练归入成长世界，产品销售和服务转化归入产品中心 |
| 全国地图 / 城市运行页 | 可作为资源连接或治理实验室辅助展示 | 暂不作为核心一级入口 |
| 共建提交页 | 用户识别与资源连接系统的后续动作入口之一 | 重构为画像识别后的投稿、需求、参访、讲课、学习申请入口 |

| 当前路由 | 当前用途 | 目标归属 | 处理方式 |
