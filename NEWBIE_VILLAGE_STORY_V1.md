# 信托制世界 · 新手村 V1 剧本

## 1. 世界定位

新手村是信托制世界的统一入口。

它不是卖课页面。

它不是培训入口。

它不是介绍信托制的宣传页。

它是所有用户进入信托制世界的共同开场。

所有角色都可以免费进入新手村：

- 业主
- 业委会成员
- 社区书记
- 街道工作人员
- 物业项目经理
- 物业企业负责人
- 创业者
- 顾问
- 研究者

完成新手村后，用户获得：

```text
【信托制正式冒险者】
```

随后根据自己的目标进入不同发展路径。

## 2. 最高叙事原则

新手村的核心叙事不是：

```text
信托制很好，所以你应该相信它。
```

而是：

```text
用户先看见旧世界的问题，
再探索各种解决办法的局限，
最后亲自进入一个运行良好的未来社区，
由此自己推导出：
真正要改变的是治理结构。
```

信托制不是平台强行灌输给用户的答案。

信托制是用户在新手村探索过程中逐步推导出来的答案。

## 3. 旧世界叙事原则

第一关和第二关讲的是旧的包干制世界。

叙事必须使用“结构性矛盾”视角。

禁止将任何群体写成坏人。

禁止使用以下表达：

- 坏物业
- 坏业主
- 坏业委会
- 不作为社区
- 物业偷钱
- 业主恶意不交费
- 业委会主任天然有私心

正确表达是：

```text
不是某个物业公司、某个业委会、某个业主是坏人。

而是旧的治理结构扭曲了权责关系，导致矛盾不断发生。
```

物业公司有无奈。

业主有委屈。

业委会有压力。

社区有困难。

真正的问题不在人，而在结构。

## 4. 新手村总结构

新手村分为三大主线关卡：

```text
第一关：旧世界困境
↓
第二关：治理议事厅
↓
第三关：信托之城
↓
新手村通关与目标分流
```

## 5. 第一关：旧世界困境

### 5.1 关卡定位

```text
认知觉醒关
```

### 5.2 任务目标

让用户看见：

```text
传统物业矛盾为什么反复发生。
```

用户需要先理解：

```text
问题不是某一个人不努力。
问题是旧结构让所有人都很难。
```

### 5.3 场景开场

天空灰暗。

业主群消息不断闪烁。

投诉电话此起彼伏。

社区办公室堆满调解记录。

物业办公室灯火通明。

系统旁白：

```text
欢迎来到传统物业矛盾丛生的旧世界。

这里，业主不信任物业。

物业抱怨收费困难。

社区疲于处理矛盾。

业委会常常陷入争议。

所有人似乎都不满意。

但所有人又都找不到真正的答案。

你将踏上一段探索之旅。

看见问题。
寻找答案。
进入未来。
```

### 5.4 NPC 一：业主

业主 NPC 表达：

```text
我每年都交物业费。

可钱到底花到哪里去了？

公共收益有多少？

维修为什么总是扯皮？

为什么我对物业越来越不信任？
```

叙事说明：

不要把业主写成恶意不交费。

业主的不信任来自：

- 不透明
- 不了解
- 缺少参与渠道
- 看不到钱的流向
- 看不到监督机制

业主不是天然对立。

业主是在旧结构中逐渐失去信任。

### 5.5 NPC 二：社区书记

社区书记 NPC 表达：

```text
投诉越来越多。

调解越来越累。

物业、业主、业委会反复冲突。

社区一直在协调。

但矛盾总是重复出现。
```

叙事说明：

不要写社区不作为。

社区面临的是：

- 资源有限
- 职责有限
- 精力有限
- 专业能力有限
- 无法长期承担专业物业服务

社区不是不愿意解决。

社区是被大量重复矛盾拖住。

### 5.6 NPC 三：物业项目经理

物业项目经理 NPC 表达：

```text
收费率越来越低。

服务要求越来越高。

人工成本越来越高。

责任越来越重。

物业人也很委屈，也很辛苦。
```

叙事说明：

不要写物业公司只想赚钱。

物业在旧结构中也被困住：

- 收费难
- 投诉多
- 成本高
- 责任重
- 服务价值难以被看见
- 一线人员长期承压

物业不是天然站在对立面。

物业也被旧结构消耗。

### 5.7 玩家任务

系统提问：

```text
你认为小区矛盾频发的根源是什么？
```

交互方式：

- 开放讨论
- 开放留言
- 开放投票

可选方向：

- 资金不透明
- 权责不清晰
- 监督机制缺失
- 沟通机制不足
- 信任持续流失

### 5.8 第一关通关结论

系统总结：

```text
经过大量实践观察发现：

表面上是人与人的矛盾。

实际上是：

资金不透明。
权责不清晰。
监督机制缺失。

最终导致信任不断流失。
```

## 10. 第四扇门：酬金制

### 10.1 场景进入

门上写着：

```text
酬金制
```

系统提示：

```text
酬金制并不是没有价值。

它已经解决了一部分问题。
```

### 10.2 酬金制的价值

酬金制让：

- 物业收入更加清晰
- 利益关系更加透明
- 一部分项目中的矛盾有所缓解
- 业主对物业服务成本有了更多理解

### 10.3 继续探索的问题

但新的问题出现：

- 谁来监督？
- 谁保证公开透明？
- 谁保障持续运行？
- 业委会是否足够专业？
- 如果业委会换届怎么办？
- 如果业委会失能怎么办？
- 如果业委会主任存在私心怎么办？

### 10.4 对业委会的叙事原则

不能妖魔化业委会。

必须明确：

```text
大多数业委会成员是义务奉献。

他们付出时间和精力。

他们也经常被误解、被质疑、被攻击，产生无力感。
```

但也必须承认：

```text
在某些制度条件下，
如果少数人掌握过多财权和决策权，
也可能出现以权谋私、能力不足、治理失衡等风险。
```

### 10.5 系统总结

```text
酬金制解决了部分问题。

但不是所有小区都具备稳定运行酬金制的条件。
```

### 15.3 正确信托制逻辑

注意：

不要写“物业不碰钱”。

这是错误的。

正确信托制逻辑是：


- 资金使用有规则
- 预算有约定
- 支出有依据
- 过程可公开
- 业主可查询
- 监督有机制

## 28. 剧本评审答案

### 28.1 为什么旧世界矛盾不断？

因为旧的治理结构中，资金不透明、权责不清晰、监督机制缺失，导致信任不断流失。

### 28.2 为什么不能归咎于某一个坏人？

因为物业公司有无奈，业主有委屈，业委会有压力，社区有困难。很多时候大家都在努力，但错误的治理结构仍然不断制造不信任。

### 28.3 为什么业主自管、社区兜底、国企接管、酬金制、反复换物业都不能彻底解决？

因为它们都能解决部分问题，但无法在所有小区中稳定解决专业化、连续性、责任承担、组织能力、监督能力和长期运行机制问题。换人不等于换结构。

### 28.4 为什么根源是治理结构问题？

因为反复出现的矛盾不是单点事件，而是资金、权责、监督、专业服务和公共参与之间的关系被旧结构扭曲后产生的系统性结果。

### 28.5 信托制为什么是一种治理结构创新？

因为它重新理顺了旧包干制中被扭曲的权责关系：业主是共同利益最终受益人，物业是受托服务人，资金按照合同和预算执行，公共收益公开透明，权责边界更清晰，各方共同监督，专业的人做专业的事。

### 28.6 新手村结束后，不同用户下一步去哪里？

- 业主：推动自己的小区了解信托制
- 业委会成员：推动小区探索信托制
- 社区书记 / 街道工作人员：为辖区小区生成治理提升建议，申请诊断、培训、参访、城市共建
- 物业项目经理：进入金牌管家成长世界
- 物业企业负责人：进入信托制经营成长路径
- 创业者：进入信托制创业成长路径
