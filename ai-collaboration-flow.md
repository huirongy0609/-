# AI Collaboration Flow

运行层：AI 协同治理流

目标：定义治理智能体之间的协同路径。

## 1. 标准智能流

```text
治理信号进入
→ 风险感知 Agent
→ AI 治理研判 Agent
→ 政策解析 Agent
→ 社区协同 Agent
→ 案例学习 Agent
→ 未来推演 Agent
→ 治理建议输出
→ 城市记忆沉淀
```

## 2. 协同机制

### 信号传递

每个 Agent 处理后，将结构化信号传给下一个 Agent。

### 记忆回写

Agent 的处理结果进入城市记忆层。

### 案例反哺

案例学习 Agent 的结果反哺风险识别和治理研判。

### 未来推演

未来推演 Agent 为协同建议提供时间维度。

## 3. 可视化表达

推荐：

- 神经连接线。
- 数据光点迁移。
- 节点缓慢脉冲。
- 处理状态环。
- 低透明记忆回流线。

禁止：

- 普通流程图。
- PPT 箭头链路。
- 聊天式步骤。
- 机械工作流图。
