# HOME_PAGE_CONCEPT_B

方案B：成长驱动型首页

关键词：

```text
成长
闯关
陪伴
成就
```

参考：

```text
Duolingo
Coursera
成长地图
```

强调：

```text
金牌管家成长世界
```

## 1. 首页首屏布局

首屏直接建立“人成长”的心智。

```text
顶部导航
聚道研究院 / 内容中心 / 制度中心 / 成长世界 / 产品中心 / 资源连接 / 治理实验室

--------------------------------------------------------------------------------

左侧主叙事

信托制社区治理平台

传播信托理念，沉淀治理经验，培养共治人才

成为更懂信托制、更会沟通、更能协同解决问题的金牌管家。

[ 开始我的成长路径 ]   [ 先了解信托制 ]

--------------------------------------------------------------------------------

右侧主视觉

金牌管家成长路径

新手村 → 一星管家 → 二星管家 → 三星管家 → 四星管家 → 五星管家

每个节点下方轻提示：
理念理解 / 公开透明 / 协同沟通 / 复杂问题 / 案例复盘 / 带教输出

底部轻提示

今日推荐任务：公共收益说明模拟
AI陪练：居民沟通场景
推荐案例：停车治理样板间
```

## 2. 首页整体结构

```text
第一屏：金牌管家成长路径主视觉
第二屏：你现在处在哪个成长阶段
第三屏：今日推荐成长任务
第四屏：真实治理案例训练
第五屏：制度工具与学习材料
第六屏：AI陪练与AI考官
第七屏：成长档案与勋章体系
第八屏：产品中心与训练营入口
第九屏：用户识别与资源连接入口
```

## 3. 用户浏览路径

```text
进入首页
  ↓
看到成长路径
  ↓
选择自己的当前阶段
  ↓
获得推荐任务
  ↓
学习相关内容和制度
  ↓
进入案例训练或AI陪练
  ↓
沉淀成长档案
  ↓
连接训练营、专家或项目资源
```

用户的第一动作是“找到自己的位置”。

## 4. 视觉气质

整体气质：

```text
温暖
陪伴
清晰
有方向
有长期成长感
```

视觉方式：

- 使用高级灰调青绿色作为成长路径主线。
- 节点可有轻微层级，但不游戏化。
- 每个成长阶段像一张温和的任务卡，而不是课程商品。
- 页面可以比方案A更有亲和力。
- 插画可使用简洁人物、文件、社区场景，但避免卡通幼稚。

## 5. 为什么符合平台定位

平台定位包含：

```text
培养共治人才
```

方案B最突出“培养共治人才”。

它让用户第一眼感受到：

- 这里不是只给我看内容。
- 这里能让我成长。
- 这里能训练我解决真实治理问题。
- 这里有一条从新手到金牌管家的路径。

它适合把“金牌管家成长世界”确立为平台核心差异化资产。

## 6. 优点

- 成长心智强。
- 容易形成用户留存。
- 有清晰的阶段感和任务感。
- 能把内容、制度、案例转化为训练材料。
- 有利于未来认证、训练营、AI陪练和成长档案。

## 7. 风险

- 容易被误解为课程平台。
- 容易滑向知识付费或训练营销售页。
- 如果闯关感过强，会破坏专业可信气质。
- 首页可能弱化“内容中心”和“制度中心”的平台感。
- 对视觉把控要求高，不能过度游戏化。

## 8. 禁止偏移

本方案禁止：

- 做成课程商城。
- 做成游戏闯关页面。
- 做成考试认证系统首页。
- 做成训练营销售页。
- 用夸张徽章、排行榜、积分刺激用户。
